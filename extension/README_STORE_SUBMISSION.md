# 🛍️ Step-by-Step Store Publishing Guide for Microsoft Edge & Mozilla Firefox

This guide contains exact instructions to publish **Ujjawal Groups Website Audit AI** to the **Microsoft Edge Add-ons Store** and **Mozilla Firefox Add-ons (AMO)**.

---

## 1. 🔵 How to Publish on Microsoft Edge Add-ons Store (FREE)

### Step 1: Create Microsoft Developer Account
1. Go to the [Microsoft Partner Center Developer Dashboard](https://partner.microsoft.com/en-us/dashboard/microsoftedge).
2. Sign in with your Microsoft account.
3. Complete registration (Microsoft Edge developer registration is **100% FREE**).

### Step 2: Create a New Extension Submission
1. Click **"Create new extension"**.
2. Upload the zip package located at:  
   `C:\Users\krish\website audit ujjawal groups\extension\dist\ujjawal-groups-website-audit-edge-v2.0.zip`
3. Fill in the Store Details:
   - **Name:** `Ujjawal Groups Website Audit AI`
   - **Short Description:** `5-Pass Empirical Website Quality Index (WQI) Audit Engine & 25-Page Handbook Implementation by Ujjawal Sharma.`
   - **Detailed Description:**  
     `Ujjawal Groups Website Audit AI evaluates any website across 15 WAEF v2.0 parameters (100 marks total) with 1-click active tab auditing, 5-sample latency benchmarking, WCAG Level AA accessibility verification, and PDF export.`
   - **Category:** `Developer Tools / Productivity`
   - **Privacy Policy URL:** `https://ujjawal-group-website-audit.vercel.app/`
   - **Support Contact / Website:** `https://github.com/Ujjawal07msd`
4. Upload Extension Logos & Screenshots:
   - Icon: Upload `extension/icons/icon128.png`
   - Screenshots: 1280x800 px screenshot of the app running.
5. Click **"Publish"**. (Review takes ~1 to 3 business days).

## 2. 🦊 How to Publish on Mozilla Firefox Add-ons (AMO) (FREE)

### Step 1: Create Firefox Developer Account
1. Go to the [Firefox Add-ons Developer Hub](https://addons.mozilla.org/en-US/developers/).
2. Sign in or create a Firefox Account (Registration is **100% FREE**).

### Step 2: Submit Your Add-on
1. Click **"Submit a New Add-on"**.
2. Select **"On this site"** (so users can download it directly from AMO).
3. Upload the Firefox zip package located at:  
   `C:\Users\krish\website audit ujjawal groups\extension\dist\ujjawal-groups-website-audit-firefox-v2.0.zip`
4. Fill in the Store Listing Details:
   - **Name:** `Ujjawal Groups Website Audit AI`
   - **Summary:** `1-Click 5-Pass Empirical Website Quality Index (WQI) Audit Engine by Ujjawal Sharma.`
   - **Homepage:** `https://ujjawal-group-website-audit.vercel.app/`
   - **Category:** `Developer Tools`
5. Click **"Submit for Review"**. (Approval usually takes ~1 to 24 hours).

## 💻 How to Test Locally Right Now (Developer Mode)

### On Microsoft Edge / Chrome / Brave:
1. Open Edge and navigate to `edge://extensions/`
2. Enable **"Developer mode"** (toggle switch in bottom left or top right).
3. Click **"Load unpacked"**.
4. Select the folder: `C:\Users\krish\website audit ujjawal groups\extension`
5. Click the extension icon in your browser toolbar to audit any website!

### On Mozilla Firefox:
1. Open Firefox and navigate to `about:debugging#/runtime/this-firefox`
2. Click **"Load Temporary Add-on..."**.
3. Select `C:\Users\krish\website audit ujjawal groups\extension\manifest_firefox.json`.
4. Click the extension icon in Firefox to run active tab audits!
